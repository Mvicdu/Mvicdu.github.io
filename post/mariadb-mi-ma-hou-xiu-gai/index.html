Mariadb忘记密码后修改密码

2019-04-17 1 min read # Linux

第一步：stop mariadb
Systemctl stop mariadb
第二步：打开跳过权限进程
/usr//bin/mysqld_safe –user=mysql –skip-grant-tables &
这个命令运行后会产生一个进程
第三步：修改密码
Mariadb的密码在mysql数据库中修改
Update user set password = password(‘xxxxx’) where user = ‘root’
注：xxxxx为想要修改的密码。
第四步：停止跳过权限进程 进程名“ mysql_safe”
第五步：start mariadb
完。
