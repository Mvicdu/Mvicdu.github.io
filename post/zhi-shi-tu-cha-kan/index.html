辅助管理之视图查看

2019-05-06 2 min read # oracle

select * from vdatabase; #可以查看数据库信息，其中可以查看数据库是否处于归档模式 另推荐使用命令： select name,log_mode from vdatabase; 查看是否为归档模式

select * from vinstance ； #查看实例；可以查看实例状态 select database_status from vinstance; #查看实例；可以查看实例状态

select * from v$controlfile ; #查询控制文件位置

select * from vdatafile; select * from dba_data_files; #这两个select语句都可以查询数据文件位置 .vdatafile 是属于动态性能视图（动态数据字典），这类视图在数据库运行时会不断进行跟新
#dba_data_files属于静态数据字典中的视图，静态数据字典有三种类型，分别为：user_、 all_、 dba_*，要想访问这些视图必须具有相关权限
ps :也可以直接查询name元素得出文件位置，例如：select name from v$datafile;
