alter system与alter database的区别

2019-07-08 1 min read

alter database //改变数据库的属性，是物理的改变。可以看得见的
alter system //改变实例的属性，是逻辑性的改变，看不见的
alter database mount 和alter database open
alter database mount //数据库已经加载了控制文件，但普通用户无法访问，只有在特权用户可以访问alter database open // 数据库一进打开，普通用户可以访问
