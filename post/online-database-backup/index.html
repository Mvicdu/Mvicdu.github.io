联机备份（online database backup）

2019-08-26

perform an online database backup steps：
1.alter database begin backup； #进入备份模式
2.Back up the database datafiles
3.alter database end backup; #一定要结束备份模式，不然scn号会一直冻结
4.alter system switch logfile;
5.Back up the archive log files.
