查看CDB数据文件目录

2019-09-09

查看CDB数据文件的存放位置，需要切换到相应的容器下。
例如，查看某个PDB数据文件的存放目录，可以通过alter session set container=pdbname命令切换到相应的PDB下
然后执行命令查看:select file_name from dba_data_files;

或者直接通过CONTAINER子句来查询:
如“select file_name from containers（dba_data_files） where con_id=3" ##这里的容器ID（con_id）可通过v$pdbs视图或show pdbs命令进行查看。
