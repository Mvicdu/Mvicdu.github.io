sqlplus窗口下支持上下翻页

2019-05-15

1、安装rlwrap包，可以使用yum install进行安装：
yum install -y rlwrap
或者下载rlwrap上传安装
例：
如上传为rpm格式
rpm -ivh rlwrap-0.42-1.el6.i686.rpm --nogpgcheck

2、在.bash_profile中配置别名，方便后续登陆
.bash_profile 位置可以用find命令查找
find /home -name .bash_profile
文件位置一般在/home/oracle/.bash_profile
添加如下命令：

alias sqlplus='rlwrap sqlplus'
alias rman='rlwrap rman'
